## Design System

### Colors
- **Background:** Neutral-950, Neutral-900 gradients
- **Borders:** Neutral-800/50 default, Amber-400/30 on hover
- **Text:** White (headings), Neutral-300-400 (body)
- **Accents:** Amber-400 to Amber-500 gradients
- **Hover:** Amber-400/30 borders with subtle glow shadows

### Typography
- **Headings:** 5xl-7xl, font-light
- **Body:** lg-xl, font-light, improved line-height
- **Spacing:** Increased padding (p-10 instead of p-8)
- **Better hierarchy:** Clear size differences between elements

### Animations
- **Duration:** 500-700ms (slower, more elegant)
- **Easing:** easeInOut for smooth transitions
- **Effects:** Subtle scale (1.03 instead of 1.05), gentle glows
- **Purposeful:** Every animation serves a UX purpose

### Buttons & CTAs
- **Primary:** Gradient from amber-400 to amber-500
- **Secondary:** Gradient backgrounds with borders
- **Hover:** Scale 1.03, shadow glow effects
- **Border radius:** Rounded-full for modern look

## Performance Improvements
- Reduced number of animated elements by ~90%
- Simplified animation calculations
- Cleaner DOM structure
- Better rendering performance
- Improved accessibility with better contrast
